What is KView?

KView is a standalone desktop application to interact with your Kubernetes cluster, some features include:

  • Filter and Search by namespace and application (pod)

  • On Demand Refresh of applications (pods)

  • Status Information application (pod), annotations, labels, events, cluster-context

  • Export YAML application (pod)

  • Container Logs

  • Pod Exec execute commands on containers

  • more to come...

Go Programming Language

KView is written entirely in Go.

Fyne Toolkit

KView utilizes the Fyne toolkit, a free and open source platform for building graphical applications for desktop, mobile and beyond.
